# Contributing to PocoOptimize

Thank you for your interest in contributing to PocoOptimize! Here are the guidelines for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a new branch for your changes
4. Make your changes
5. Test thoroughly
6. Submit a pull request

## Code of Conduct

Be respectful and constructive in all interactions.

## Reporting Issues

- Use GitHub Issues to report bugs
- Include device model, Android version, and steps to reproduce
- Provide relevant logs if applicable

## Pull Requests

- Keep changes focused and minimal
- Write clear commit messages
- Update documentation as needed
- Test on actual device before submitting

## Testing

- Test on Poco F6 or compatible devices
- Test both installation and uninstallation
- Verify no bootloops or crashes
- Check thermal management works correctly
